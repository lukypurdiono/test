PERT.5 BEKERJA DENGAN GIT

Tata cara bekerja dengan git

  • Masuk ke git bash
  • Masuk ke folder yang akan dijadikan repo(cd -> change directory)
  • Beberapa perintah yang ada di GIT yang sering digunakan
    1. GIT init -> untuk menjadikan folder sebagai folder master (REPO)
    2. GIT add file -> untuk menambahkan file (git add index.html)
    3. GIT Status -> Untuk mengetahui status git kita, apakah sudah di commit atau belum
    4. GIT Commit -m -> untuk meng commit file yang ada (git commit -m "Menambahkan index.html"
    5. git config --global user.email "lukypurdiono@gmail.com"
    6. git config --global user.name "Luky Purdiono"
    7. Git add. -> menambahkan semua file yang berubah
    8. GIT Log -> untuk mengetahu apa saja yang sudah kita lakukan
    9. GIT Checkout -> untuk mengembalikan file yang terhapus/hilang (git checkout (5 digit pertama hash--nama file) (git checkout 54213 --style.css)
    10. GIT commit -am "message" -> shortcut untuk menambahkan sekaligus mengcommit file
  • Alur GIT WORKING TREE->STAGING AREA->COMMIT

PERT.6 BRANCH AND MERGE

  • GIT Branch namabranch-> untuk menambahkan branch
  • GIT branch -> untuk mengetahui jumlah branch
  • GIT log --all --decorate --oneline --graph -> untuk mengetahui graph yang sudah terbentuk
  • alias graph=GIT log --all --decorate --oneline --graph -> untuk menyingkat command
  • GIT checkout namaBranch-> untuk berpindah branch
  • git merge namaBranch -> untuk menggabungkan branch
  • git branch -d namaBranch -> untuk mengapus branch yang sudah di merge
  • git branch -D namaBranch -> untuk menghapus paksa branch
  • git branch --merged -> untuk mengetahui branch yang sudah di merged
