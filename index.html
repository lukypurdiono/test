BEKERJA DENGAN GIT

Tata cara bekerja dengan git

  • Masuk ke git bash
  • Masuk ke folder yan akan dijadikan repo(cd -> change directory)
  • Beberapa perintah yang ada di GIT yang sering digunakan
    1. GIT add file -> untuk menambahkan file (git add index.html)
    2. GIT Status -> Untuk mengetahui status git kita, apakah sudah di commit atau belum
    3. GIT Commit -m -> untuk meng commit file yang ada (git commit -m "Menambahkan index.html"
    4. git config --global user.email "lukypurdiono@gmail.com"
    5. git config --global user.name "Luky Purdiono"
    6. Git add. -> menambahkan semua file yang berubah
